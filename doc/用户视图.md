# 普通用户视图
1. 我要报修
2. 我的
    1. 我的信息
        1. 真实姓名
        2. 电话
        3. 地址

    2. 我的报修
        1. 待审核
        2. 待维修
        3. 待确认
        4. 全部
    3. 设置、用户反馈、关于等
3. 报单详情页
    1. 维修主体
    2. 维修类型
    3. 故障描述
    4. 联系电话
    5. 其他信息（报价、报修状态等）
    6. 操作按钮（提交、确认、拒绝等）
